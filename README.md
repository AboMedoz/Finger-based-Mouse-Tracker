# Finger-based-Mouse-Tracker

Using your Webcam and Fingers track and control your Mouse 

---
## Controls
#### **Track**: Index Finger
#### **Click**: Pinch Index + Thumb 
#### **Scroll**: 
    Up : Fingers Opened
    Down: Fingers Closed
#### **Quit**: q

---
## Building
```shell
pyinstaller main.spec
```
